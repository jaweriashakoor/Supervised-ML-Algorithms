A comprehensive exploration of Supervised Machine Learning. This repository serves as a technical portfolio for implementing algorithms that solve both **Classification** (categorical) and **Regression** (numerical) problems using Python and Scikit-Learn.


## 🏗️ Repository Architecture

The codebase is structured to showcase mastery over different families of algorithms:

### 📈 Regression (Predicting Continuous Values)
* **Linear Regression (`linear.py`):** Predicting trends such as house prices and fish measurements.
* **KNN Regression (`knn.py`):** Using distance-based logic for numerical prediction.

### 🧪 Classification (Categorical Prediction)
* **Logistic Regression (`logistic_regression.py`):** Binary classification for medical risks (Diabetes) and spam detection.
* **Naive Bayes (`naive_bayes.py`):** Probabilistic classification for text and medical data.
* **Support Vector Machines (SVM):** Finding the optimal hyperplane for complex data boundaries.

### 🌳 Tree-Based & Ensemble Methods
* **Decision Trees:** Hierarchical rule-based modeling.
* **Random Forest:** Using ensembles to increase accuracy and reduce overfitting in burnout and addiction analysis.


## 🛠️ Technical Workflow

1.  **Data Preprocessing:** Cleaning real-world datasets (Mental Health, Smartphone Usage, etc.).
2.  **Feature Selection:** Identifying which variables drive the most predictive power.
3.  **Model Training:** Splitting data into Training/Testing sets (typically 80/20 or 70/30).
4.  **Evaluation:** Using metrics like Accuracy, Precision, and R-Squared to validate results.
5.  **Visualization:** Generating plots (see `my_machine_learning_plot.png`) to interpret model behavior.
